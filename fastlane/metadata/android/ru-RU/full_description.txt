XiVPN — это графический клиент для XTLS/Xray-Core, предоставляющий прокси-протоколы для обхода цензуры.

Возможности:
- Получение прокси по ссылкам на подписку
- Простой и удобный графический интерфейс
- Настраиваемые параметры прокси
- Маршрутизация
- Цепочки прокси
- Полностью открытый исходный код

Поддерживаемые протоколы:
- VLESS
- VMESS
- Shadowsocks
- Trojan
- Wireguard

Поддерживаемые транспортные протоколы:
- Websocket
- QUIC
- Httpupgrade
- XHTTP

Поддерживаемые протоколы безопасности:
- TLS / uTLS
- REALITY
